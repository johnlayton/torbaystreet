## json-query

### Setup oh-my-zsh

```zsh
pushd $ZSH/custom/plugins && \
  git clone git@github.com:johnlayton/torbaystreet.git json-query && \
  popd || echo "I'm broken"
```

```zsh
plugins=(... recruitment)
```

### Setup other

```zsh
pushd $HOME && \
  git clone git@github.com:johnlayton/torbaystreet.git .json-query && \
  popd || echo "I'm broken"
```

```zsh
source ~/.json-query/json-query.plugin.zsh
```
